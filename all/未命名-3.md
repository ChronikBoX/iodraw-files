```mermaid
graph TD
    start[开始] --> check[检查I2C总线空闲]
    check --> send[发送测量命令]
    send --> wait[等待转换完成]
    wait --> read[读取温湿度原始数据]
    read --> crc{CRC校验通过?}
    crc -->|是| convert[转换为物理量]
    crc -->|否| reuse[复用上一次有效值]
    convert --> push[数据推入滑动窗口]
    reuse --> push
    push --> calc[计算窗口均值]
    calc --> update[更新全局温湿度值]
    update --> end[结束]
```
